
# Full-Stack E-Commerce Product Catalog

## Features
- Modular Next.js architecture
- Client-side routing
- Optimized image loading
- Responsive UI
- Ready for Vercel deployment

## Installation

```bash
npm install
npm run dev
```

## Build for Production

```bash
npm run build
npm start
```

## Deploy to Vercel

1. Push to GitHub
2. Import repository into Vercel
3. Click Deploy
